Let's Talk About Me

I was born to do one thing but, I haven't been able to find a way to get paid sitting on the couch and binge watching Netflix. So, I decided to start studying Frontend Web Development. If you ask my family, they will tell you how technically savvy I am. I don't think I would be bragging to much if I said I am the go to guy when they need someone to hook up their printers. If the word of my family isn't enough keep reading and I might surprise you.

Front-end

Whether it is design, markup, or JavaScript I have the skills you need. From paragraphs to animation I can create or improve your website by creating a friendly and satisfying user experience for all of your users.

Dev Tools

Having a tool belt filled with the right tools is essential for any good tradesmen. My belt may have a few more notches than it use to but, it is still full of the best tools that any frontend developer could ask for. Atom and C9 provide me with the ability to write markup from anywhere and on any computer. By using Github as a repository I can organize and control versions of all the code I have written.

Frameworks

Building great sites takes a lot of work if you don't know the right frameworks. That is why I use Bootstrap and Foundation. Using these frameworks allows me to produce amazing sites that your users will love in a fraction of the time.
